4.6.3 sub 将匹配到的数据进行替换

4.6.3.1 将匹配到的阅读次数加1

1) 方法1

示例代码：

import re

ret = re.sub(r"\d+", '998', "python = 997")
print(ret)

执行结果：

2) 方法2

示例代码：

import re

def add(temp):
    strNum = temp.group()
    num = int(strNum) + 1
    return str(num)

ret = re.sub(r"\d+", add, "python = 997")
print(ret)

ret = re.sub(r"\d+", add, "python = 99")
print(ret)

4.6.4 split

split 的功能是根据匹配进行切割字符串，并返回一个列表。

需求：切割字符串"info:mike:18:shenzhen"

示例代码：

import re

#以:或者空格切割
ret = re.split(r":| ","info:mike 18 shenzhen")
print(ret)
